#### 描述
本项目为NFT铸造、空投、挖矿、盲盒售卖资源包,包括源码、脚本。NFT资产标准 `ERC1155`

#### 该项目合作案例

| 名称 | 描述 |
| --- | --- |
| 烤仔NFT | 盲盒售卖 + 挖矿 |
| Condragon NFT | 盲盒售卖 + 挖矿 |
| Conhero NFT | 盲盒售卖 + 挖矿 |

---
#### 介绍

源码包括NFT创建、NFT空投、NFT盲盒售卖、NFT质押挖矿，还有testnet的执行脚本，在接下来会介绍使用方法。

通过该脚本铸造的NFT的，简单填写申请表，便可直通[Moongaming NFT平台](https://moonswap.fi/nft)，接入非常便捷。
